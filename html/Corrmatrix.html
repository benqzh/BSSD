Corrmatrix {BSSD}R Documentation

Correlation matrix R

Description

Generate correlation matrix R of order n, with period Ty.

Usage

Corrmatrix(ph, n, Ty)

Arguments

ph

correlation coefficient rho

n

the order of the matrix

Ty

period of the seasonal effect

Value

correlation matrix R

Examples

Corrmatrix(0.8, 12, 6)
