unemployment {BSSD}R Documentation

Dataset: unemployment level

Description

U.S. unemployment level Jan 2004 - Dec 2018

Usage

data("unemployment")

Format

A data frame with 180 observations on the following 2 variables.

y

a numeric vector of unemployment level

label

a numeric vector of date

Examples

data(unemployment)
y=unemployment$y
timelab=unemployment$label
plot(y,type = 'l', lty=3,xaxt="n",ylab = '',xlab="Time (by 2 months)",ylim = c(min(y),max(y)))
at=seq(from = 1, to = length(y), by =2)
axis(1, at, labels=format(timelab[at],"%Y-%m"))
