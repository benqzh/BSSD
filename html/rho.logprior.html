rho.logprior {BSSD}R Documentation

Prior for rho

Description

This fuction computes the log-density of the prior of rho.

Usage

rho.logprior(rho, choice)

Arguments

rho

value of rho

choice

choice of prior for rho, choice=0,1,2,...

Details

When choice=0, p(rho) ~ 1;

when choice=1, p(rho) ~ (1-rho)^(-1/2),\ 0<rho<1;

when choice >= 2, p(rho) ~ (1-rho)^(-1/2)*(1+rho/(k-1)^(-1/2) , 0<rho<1,k=choice .

Value

log-density of the prior

Examples

rho.logprior(0.8, choice = 1)
## The function is currently defined as
function (rho, choice)
{
    if (choice == 0) {
        return(0)
    }
    if (choice == 1) {
        return(-0.5 * log(1 - rho))
    }
    if (choice > 1) {
        return(-0.5 * log(1 - rho) - 0.5 * log(1 + rho/(choice -
            1)))
    }
  }
